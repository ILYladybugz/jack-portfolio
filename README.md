### Run Locally

To run **sherie-portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/ILYladybugz/sherie-portfolio.git
```

Windows:

```bash
git clone https://github.com//ILYladybugz/sherie-portfolio.git
```
